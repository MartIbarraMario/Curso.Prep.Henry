Objetos en javascript

Objetos:

Las variables en JavaScript son contenedores para valores y datos. En esencia, losobjetos también funcionan cómo variables, con la diferencia de que estos pueden almacenar muchas variables.

Piensa en los objetos como una lista de valores que son escritos de la siguiente manera: name:value (nombre:valor), con el nombre y los valores separados por dos puntos.

Ejemplo de sintaxis de un Objeto:

var Persona = {
nombre: 'Mario',
edad: 25,
genero: 'masculino'
}

Propiedades:

Una propiedad es la unión entre un nombre (o clave) y un valor.
El valor de una propiedad puede ser una función, en cuyo caso la propiedad es conocida como un método.

Ejemplo:

var persona = {
saludar: function() {
console.log('¡Hola mundo!');
}
}
Persona.saludar();

Bucle “for … in loop”:

Es ligeramente diferente, al bucle clásico "for" pero, apesar de comienzar igual, esté bucle alberga entre sus paréntesis la declaración de una variable, ademas de la palabra clave "in" y el nombre del objeto.

Esto recorrerá cada clave del objeto y finalizará cuando se haya revisado todas las claves.

Notación de puntos vs notación de corchetes:

Tanto la notación de puntos, como la notación de corchetes se emplean para acceder a los valores.

1.-Notación de puntos:
persona.nombre;

2.- Notación de corchetes:
persona['Mario'];

Este ultimo se emplea al igual que llamar a un elemento dentro de un arreglo, pero entre los corchetes debemos usar una cadena de texto, o número, o una variable que apunte a una cadena o número.
