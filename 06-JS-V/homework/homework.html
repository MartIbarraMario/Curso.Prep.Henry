Breve introducción a los conceptos básicos de Javascript

Prototype:

Dentro de Javascript, todos los objetos tienen como referencia una platilla o estructura estandarizada.
No solo con la finalidad de ahorrar memoria en la computadora, si no que, dicho prototipo funciona cómo fuente principal para la creación de cualquier objeto.

Constructors:

La función constructora o "constructor" se debe entender cómo un metodo especial que nos permite crear e iniciar la estructura principal de un objeto creado a partir de una clase.
